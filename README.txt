Simulation 1 contains the code for the simulation study without clustered treatment allocation.
Simulation 2 contains the code for the simulation study with clustered treatment allocation.

Application to Cambodia Dataset contains the file for the Cambodia Data along with the cleaned dataset. Methodology is contained within the file, Application to Cambodia Dataset, and has files where different combinations of methodlogy, crossfitting, and spatial accountance are then applied to the Cambodia deforestation dataset. All files are:
- Data_Analysis_Clean.R (code for cleaning the data)
- Data_Analysis_DID.R
- Data_Analysis_OLS.R
- Data_Analysis_DML_Cross_LatLong_noRE.R
- Data_Analysis_DML_Cross_Wend_noRE.R
- Data_Analysis_DML_Cross_noSpatial_noRE.R
- Data_Analysis_DML_noCross_LatLong_noRE.R
- Data_Analysis_DML_noCross_Wend_noRE.R
- Data_Analysis_DML_noCross_noSpatial_noRE.R
